# NutriMax Documentation

Welcome to the official technical documentation for **NutriMax**. 

NutriMax is a web platform for nutritional coaching built with simplicity, clean code, and maintainability in mind. We strictly avoid over-engineering, using a lightweight MVC architecture on the backend and pure Vanilla JS on the frontend.

## Documentation Structure

- [Getting Started](01-getting-started.md): Quick overview of the project and its goals.
- [Installation Guide](02-installation.md): Step-by-step instructions to set up the local development environment.
- [Reference / API](reference/api.md): Deep dive into the backend structure and API endpoints.

## Core Philosophy

- **Zero Assumptions**: Clean separation of concerns.
- **Vanilla First**: No heavy frontend frameworks. Native browser APIs, Service Workers, and CSS variables.
- **MVC & API-Driven**: A pure PHP backend serving both static HTML views and RESTful JSON APIs.
- **Scalable**: Built to perform and scale realistically.
