# Documentación de NutriMax

Bienvenido a la documentación técnica oficial de **NutriMax**. 

NutriMax es una plataforma web para coaching nutricional construida pensando en la simplicidad, el código limpio y la mantenibilidad. Evitamos estrictamente la sobreingeniería, utilizando una arquitectura MVC ligera en el backend y JavaScript Vanilla puro en el frontend.

## Estructura de la Documentación

- [Guía de Inicio](01-getting-started.md): Resumen rápido del proyecto y sus objetivos.
- [Guía de Instalación](02-installation.md): Instrucciones paso a paso para configurar el entorno de desarrollo local.
- [Referencia / API](reference/api.md): Profundización en la estructura del backend y los puntos de acceso de la API.

## Filosofía Principal

- **Zero Asunciones**: Clara separación de responsabilidades.
- **Vanilla First**: Sin frameworks pesados en el frontend. APIs nativas del navegador, Service Workers y variables CSS.
- **MVC & API-Driven**: Un backend en PHP puro que sirve tanto vistas HTML estáticas como APIs JSON RESTful.
- **Escalable**: Construido para rendir y escalar de forma realista.
