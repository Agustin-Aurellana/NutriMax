# NutriMax - Bitácora de Desarrollo 🚀

Organización lógica del desarrollo por semanas.

---

### 📅 Semana 1: Cimientos y Lógica Base
*Enfoque: Estética inicial y motores de cálculo.*
- **Naming:** Elección de **NutriMax**.
- **Diseño:** Paleta original de verde y blanco.
- **Perfil de Usuario:** Implementación del modelo de datos inicial (Edad, Sexo, Actividad).
- **Cálculo Nutricional:** Motor base con la fórmula de **Mifflin-St Jeor** para TDEE y Macros.

### 📅 Semana 2: Arquitectura y Branding
*Enfoque: Identidad visual y navegación.*
- **Logo:** Diseño definitivo y variantes para PWA.
- **UI Base:** Creación del Dashboard y la Barra de Navegación Lateral (Sidebar).
- **Registro Básico:** Primer prototipo de la tabla de registros diarios.

### 📅 Semana 3: El Giro "Premium" e IA
*Enfoque: Experiencia de usuario y vanguardia.*
- **Rediseño:** Cambio radical a paleta **Royal Indigo & Emerald** (Azul/Morado).
- **IA (V1):** Primera integración experimental con Gemini API.
- **Modo Oscuro:** Implementación inicial (Midnight Dark).
- **Recetario:** Base de datos semilla (Seed) con recetas saludables básicas.

### 📅 Semana 4: Estabilidad y Potencia de IA
*Enfoque: Optimización y corrección de errores.*
- **IA Definitive:** Migración a **Groq (Llama 3.3)** para respuestas instantáneas.
- **Doble Tema:** Habilitación formal de **Modo Claro** y **Modo Oscuro**.
- **Motores de Búsqueda:** Primera integración con Edamam y USDA para búsqueda de alimentos globales.
- **Fixes de Lógica:** Corrección de la persistencia de datos y estados compartidos.

### 📅 Semana 5: Globalización y UX Final (S5 actual)
*Enfoque: Utilidades avanzadas y sincronización.*
- **Localización:** Detección de país vía Nominatim para alimentos regionales.
- **Utilidades:**
  - Rastreador de **Hidratación** (Vasos de agua).
  - Historial de **Peso** con visualización gráfica.
  - Escáner de **Código de Barras** funcional.
- **Sincronización:** Sistema híbrido de guardado en **IndexedDB + LocalStorage** para evitar pérdida de datos.
